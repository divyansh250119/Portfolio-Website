# Personal Portfolio Website

A clean and beginner-friendly multi-page portfolio website built using **HTML** and **CSS**.

This project includes:

- Home page
- About Me page
- Skills page
- Contact page
- Reusable navigation bar
- Simple responsive layout and card-based sections

---

## 📁 Project Structure

```bash
portfolio-website/
│── index.html
│── about.html
│── skills.html
│── contact.html
│── styles.css
└── README.md
```

---

## 🌐 Pages Overview

### 1. Home (`index.html`)
- Welcome/hero section
- Short introduction
- Quick links to other pages

### 2. About Me (`about.html`)
- Personal background
- Education
- Experience
- Interests and goals

### 3. Skills (`skills.html`)
- Technical skills
- Tools & technologies
- Soft skills
- Learning roadmap

### 4. Contact (`contact.html`)
- Contact form (UI only)
- Email / GitHub info
- Availability details

---

## 🎨 Styling Highlights

- Global styles in **external CSS** (`styles.css`)
- Reusable **navigation bar** across all pages
- Card-style content sections
- Clean color palette and spacing
- Hover effects on nav links and buttons
- Mobile-friendly layout with flexible containers

---

## 🚀 How to Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Double-click `index.html`  
   **or** open it using VS Code + Live Server.

---

## 🧭 Navigation Flow

- Home → About Me
- Home → Skills
- Home → Contact
- All pages are connected through the top navigation bar.
